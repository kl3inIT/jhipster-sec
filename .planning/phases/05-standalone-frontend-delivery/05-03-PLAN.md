- Required: id (Long), code (String), name (String), ownerLogin (String)

Department:
- Required: id (Long), code (String), name (String)
- Optional: organization (FK -> IOrganization), costCenter (String)

Employee:
- Required: id (Long), employeeNumber (String), firstName (String), lastName (String)

- GET/POST /api/organizations — paginated list, create
- GET/PUT/DELETE /api/organizations/{id} — read, update, delete
- GET/POST /api/departments — paginated list, create
- GET/PUT/DELETE /api/departments/{id}
- GET/POST /api/employees — paginated list, create
- GET/PUT/DELETE /api/employees/{id}

All return Map<String,Object> — optional fields absent when denied.
Pagination: page, size, sort params; X-Total-Count header.

**Model (organization.model.ts):**

```typescript
export interface IOrganization {
  id?: number;
  code?: string;
  name?: string;
  ownerLogin?: string;
  budget?: number;       // permission-gated — absent when denied
}

export type NewOrganization = Omit<IOrganization, 'id'> & { id: null };
```
