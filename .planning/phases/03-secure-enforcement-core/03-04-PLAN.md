    ```java
    private SecuredEntityEntry resolveEntry(String entityCode) {
        return catalog.findByCode(entityCode)
            .orElseThrow(() -> new IllegalArgumentException("Entity code not in secured catalog: " + entityCode));
    }
    ```

    **Private helper `checkCrud(Class<?> entityClass, EntityOp op)`:**
    ```java
    private void checkCrud(Class<?> entityClass, EntityOp op) {
        CrudEntityContext ctx = new CrudEntityContext(entityClass, op);
        accessManager.applyRegisteredConstraints(ctx);
        if (!ctx.isPermitted()) {
            throw new AccessDeniedException(
                op.name() + " denied on " + entityClass.getSimpleName());
        }
    }
    ```

    **`loadByQuery(SecuredLoadQuery query)` — READ flow per D-05:**
    1. Resolve catalog entry: `SecuredEntityEntry entry = resolveEntry(query.entityCode())`
    2. Get entity class: `Class<?> entityClass = entry.entityClass()`
    3. **Step 1 — CRUD check:** `checkCrud(entityClass, EntityOp.READ)`
    4. **Step 2 — Row-policy specification:** `Specification<Object> rowSpec = rowLevelSpecificationBuilder.build(entityClass, EntityOp.READ)`
    5. **Step 3 — Query execution:**
       - Get `JpaSpecificationExecutor<Object> specRepo = repositoryRegistry.getSpecificationExecutor(entityClass)`
       - Build the combined specification: start with `rowSpec`
       - If `query.jpql()` is not null and not blank: validate `entry.jpqlAllowed()` is true, else throw `AccessDeniedException("JPQL queries not allowed for " + query.entityCode())`
       - For now, Phase 3 does not implement arbitrary JPQL-to-Specification conversion. If `query.jpql()` is provided, build a simple Specification that always passes (Phase 4 can refine this for sample entities). Log a warning if JPQL is provided but not yet converted to Specification.
       - Execute: `Page<Object> page = specRepo.findAll(rowSpec, query.pageable())`
    6. **Step 4 — Fetch plan resolution:** `FetchPlan fetchPlan = fetchPlanResolver.resolve(entityClass, query.fetchPlanCode())`
    7. **Step 5 — Attribute-filtered serialization:**
       - Map each entity in `page.getContent()` through `secureEntitySerializer.serialize(entity, fetchPlan)`
       - Return a new `PageImpl<>(serializedList, query.pageable(), page.getTotalElements())`

    **`save(String entityCode, Object id, Map<String, Object> attributes, String fetchPlanCode)` — WRITE flow per D-06:**
    1. Resolve: `SecuredEntityEntry entry = resolveEntry(entityCode)`
    2. Get entity class and determine operation: if `id` is null -> `EntityOp.CREATE`, else `EntityOp.UPDATE`
    3. **Step 1 — CRUD check:** `checkCrud(entityClass, op)`
    4. **Step 2 — Row-constrained target lookup (UPDATE only):**
       - Get `JpaSpecificationExecutor<Object> specRepo = repositoryRegistry.getSpecificationExecutor(entityClass)`
       - Build id specification: `(root, query, cb) -> cb.equal(root.get("id"), id)`
       - Build row spec: `rowLevelSpecificationBuilder.build(entityClass, EntityOp.UPDATE)`
       - Execute: `specRepo.findOne(idSpec.and(rowSpec))` — if empty, throw `AccessDeniedException("UPDATE denied — entity not found or row policy restricts access")`
       - For CREATE: instantiate new entity via `entityClass.getDeclaredConstructor().newInstance()`
    5. **Step 3 — Attribute edit enforcement:** `secureMergeService.mergeForUpdate(entity, attributes)`
    6. **Step 4 — Persistence:**
       - Get `JpaRepository<Object, ?> repo = repositoryRegistry.getRepository(entityClass)`
       - `Object saved = repo.save(entity)`
    7. **Step 5 — Secure re-read:**
       - Resolve fetch plan: `FetchPlan fetchPlan = fetchPlanResolver.resolve(entityClass, fetchPlanCode)`
       - Return `secureEntitySerializer.serialize(saved, fetchPlan)`

    **`delete(String entityCode, Object id)` — DELETE flow:**
    1. Resolve: `SecuredEntityEntry entry = resolveEntry(entityCode)`
    2. **Step 1 — CRUD check:** `checkCrud(entry.entityClass(), EntityOp.DELETE)`
    3. **Step 2 — Row-constrained target lookup:**
       - Same pattern as save UPDATE step 2 but with `EntityOp.DELETE`
       - If not found, throw `AccessDeniedException("DELETE denied — entity not found or row policy restricts access")`
    4. **Step 3 — Delete:** `repositoryRegistry.getRepository(entityClass).delete(entity)`
