  <action>
    Create `SecureEntitySerializerImpl.java` in `com.vn.core.security.serialize` as a `@Component` implementing `SecureEntitySerializer`.

    Constructor-inject `AttributePermissionEvaluator`.

    **`serialize(Object entity, FetchPlan fetchPlan)` method:**
    1. If `entity` is null, return null
    2. Get the entity class from `entity.getClass()`
    3. Create a `BeanWrapperImpl` wrapping the entity: `new BeanWrapperImpl(entity)`
    4. Create result `Map<String, Object> values = new LinkedHashMap<>()`
    5. Iterate `fetchPlan.getProperties()`:
       a. For each `FetchPlanProperty property`:
          - String `attr = property.name()`
          - **id always-visible rule per D-16:** If `isAlwaysVisible(attr)`, read the value via `wrapper.getPropertyValue(attr)` and put it in the map, skip permission check
          - Otherwise, check `attributePermissionEvaluator.canView(entityClass, attr)` — if denied, **silently skip** per D-15
          - If permitted, read value via `wrapper.getPropertyValue(attr)`
          - If `property.fetchPlan()` is not null (it's a reference/association per D-17):
            - If value is `null`, put `null`
            - If value is `Collection<?>`, map each item through recursive `serialize(item, property.fetchPlan())`, filter nulls, collect to list
            - Otherwise, recursively call `serialize(value, property.fetchPlan())`
          - If `property.fetchPlan()` is null (scalar), put the raw value

    **`isAlwaysVisible(String attr)` private helper:**
    ```java
    private boolean isAlwaysVisible(String attr) {
        return "id".equals(attr.toLowerCase(Locale.ROOT));
    }
    ```

    **Imports needed:**
    - `org.springframework.beans.BeanWrapperImpl`
    - `org.springframework.stereotype.Component`
    - `java.util.*`
    - `com.vn.core.security.fetch.FetchPlan`
    - `com.vn.core.security.fetch.FetchPlanProperty`
    - `com.vn.core.security.permission.AttributePermissionEvaluator`
