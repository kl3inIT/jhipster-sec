## Decisions Made

- Explicit PATCH endpoints reuse the existing secure save flow instead of introducing a parallel patch merge implementation.
- The permission matrix now discovers assignable apps from the menu-definition catalog as well as existing grants, so the UI can create a role's first grant in a new app.
- The brownfield verification sweep is executed with `integrationTest` under Java 25 because that is the only path that actually runs backend `*IT` suites in this repository.
