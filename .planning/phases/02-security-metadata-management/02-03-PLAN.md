    Create `SecRoleAdminResource` in package `com.vn.core.web.rest.admin.security` (per D-05, D-19).

    Class annotations:
    ```java
    @RestController
    @RequestMapping("/api/admin/sec/roles")
    @PreAuthorize("hasAuthority(\"" + AuthoritiesConstants.ADMIN + "\")")
    ```

    Inject via constructor: `AuthorityRepository authorityRepository`, `SecPermissionRepository secPermissionRepository`, `@Value("${jhipster.clientApp.name}") String applicationName`.

    Private constant: `private static final String ENTITY_NAME = "secRole";`

    **POST /api/admin/sec/roles** (`createRole(@Valid @RequestBody SecRoleDTO dto)`):
    - If `authorityRepository.findById(dto.getName()).isPresent()`, throw `new BadRequestAlertException("A role with this name already exists", ENTITY_NAME, "nameexists")`
    - Create `Authority` entity: `new Authority().name(dto.getName())`, set `displayName` from dto, set `type` from `RoleType.valueOf(dto.getType())`
    - Save via `authorityRepository.save(authority)`
    - Map back to SecRoleDTO for response
    - Return `ResponseEntity.created(new URI("/api/admin/sec/roles/" + dto.getName())).headers(HeaderUtil.createEntityCreationAlert(applicationName, true, ENTITY_NAME, dto.getName())).body(resultDto)`

    **GET /api/admin/sec/roles** (`getAllRoles()`):
    - `authorityRepository.findAll()` -> map each to SecRoleDTO
    - Return `ResponseEntity.ok(dtoList)`

    **GET /api/admin/sec/roles/{name}** (`getRole(@PathVariable String name)`):
    - `authorityRepository.findById(name)` -> map to SecRoleDTO
    - Return `ResponseUtil.wrapOrNotFound(optionalDto)`

    **PUT /api/admin/sec/roles/{name}** (`updateRole(@PathVariable String name, @Valid @RequestBody SecRoleDTO dto)`):
    - If `!name.equals(dto.getName())`, throw BadRequestAlertException "Name in path and body must match", ENTITY_NAME, "namemismatch"
    - Find authority by name, or throw 404 via `authorityRepository.findById(name).orElseThrow(() -> new BadRequestAlertException("Role not found", ENTITY_NAME, "notfound"))`
    - Update `displayName` and `type` fields only (name is PK, immutable)
    - Save and return updated DTO

    **DELETE /api/admin/sec/roles/{name}** (`deleteRole(@PathVariable String name)`):
    - Application-level cascade: `secPermissionRepository.deleteByAuthorityName(name)` first (even though DB has CASCADE, explicit delete provides audit consistency)
    - `authorityRepository.deleteById(name)`
    - Return `ResponseEntity.noContent().headers(HeaderUtil.createEntityDeletionAlert(applicationName, true, ENTITY_NAME, name)).build()`

    Manual DTO mapping helper (private method):
    ```java
    private SecRoleDTO toDto(Authority authority) {
        SecRoleDTO dto = new SecRoleDTO();
        dto.setName(authority.getName());
        dto.setDisplayName(authority.getDisplayName());
        dto.setType(authority.getType().name());
        return dto;
    }
    ```

    Add `@Transactional` on the `deleteRole` method (needs both permission delete and authority delete in one transaction). Import `org.springframework.transaction.annotation.Transactional`.

    - `secPermissionRepository.findById(id)` -> map to DTO
    - Return `ResponseUtil.wrapOrNotFound(optionalDto)`

    **PUT /api/admin/sec/permissions/{id}** (`updatePermission(@PathVariable Long id, @Valid @RequestBody SecPermissionDTO dto)`):
    - If `dto.getId() == null`, throw BadRequestAlertException "Invalid id", ENTITY_NAME, "idnull"
    - If `!id.equals(dto.getId())`, throw BadRequestAlertException "ID in path and body must match", ENTITY_NAME, "idmismatch"
    - If `secPermissionRepository.findById(id).isEmpty()`, throw BadRequestAlertException "Permission not found", ENTITY_NAME, "notfound"
    - Validate FK exists (same as create)
    - Convert, save, return updated DTO
